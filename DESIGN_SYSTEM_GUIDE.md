**كل بطاقة تحتوي على:**
- أيقونة ملونة مع خلفية زاهية

## 📏 نظام المسافات

```
الداخلي (Padding):
├─ xs: 8px   (أزرار صغيرة، أيقونات)
├─ sm: 12px  (حقول النماذج)
├─ md: 16px  (بطاقات، أقسام)
├─ lg: 20px  (رؤوس الأقسام)
└─ xl: 24px  (محتوى الصفحة)

الخارجي (Margin/Spacing):
├─ xs: 8px   (بين الأشياء الصغيرة)
├─ sm: 12px  (بين العناصر)
├─ md: 16px  (بين الصفوف)
└─ lg: 28px  (بين الأقسام الرئيسية)

الحدود (Border Radius):
├─ sm: 8px   (أيقونات صغيرة)
├─ md: 12px  (حقول النماذج)
└─ lg: 18px  (البطاقات الرئيسية)
```

```
├─ w600: 14px (النصوص العادية)

الأحجام:
├─ 11px: تعليقات، توضيحات

├─ 14px: نص عادي
├─ 16px: عناوين
├─ 18px: عناوين فرعية
├─ 20px: عناوين الصفحات
```

```dart
// على الأزرار والبطاقات

- رفع الظل: blur يزداد من 8px إلى 12px
- Cursor يتغير إلى pointer
```

### Focus States
```dart
// على حقول الإدخال
- حدود اللون الأساسي (width: 2px)
- خلفية بتدرج طفيف
- ظل ناعم بلون الأساسي
```

```dart
// عند الحفظ
- زر يعرض spinner
```
