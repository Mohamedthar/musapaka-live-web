## 🎨 نظام التصميم الموحد

### الألوان المستخدمة
```
فترة التسجيل:    #2563EB (أزرق) على #EFF6FF
فترة الاختبارات: #7C3AED (بنفسجي) على #F5F3FF
الحالة الفعالة:  #10B981 (أخضر) على #F0FDF4
الحالة الموقوفة: #EF4444 (أحمر) على #FEF2F2
```

### المكونات الموحدة
```
Headers:      gradient background + أيقونة ملونة
Toggle:       custom switch مع status badge
Date Fields:  calendar icon + date picker
Alerts:       icon + title + description
Cards:        white background + border + shadow
```

### المسافات المنتظمة
```
Padding:   12px, 14px, 18px
Spacing:   10px, 12px, 14px
Gap:       18px-32px (بين الأقسام)
```

## 🔧 كيفية التطبيق

### الخطوة 1: استبدال الملفات

في `settings_screen.dart`:

```dart
// استبدل الـ imports
import 'widgets/registration_dates_section_improved.dart';
import 'widgets/contest_info_section_improved.dart';
import 'widgets/schedule_section_improved.dart';

// استبدل في build()
RegistrationDatesSectionImproved(
  isRegistrationOpen: _isRegistrationOpen,
  onRegistrationToggle: (v) => setState(() => _isRegistrationOpen = v),
  registrationStart: _registrationStart,
  onRegistrationStartChanged: (d) => setState(() => _registrationStart = d),
  // ... بقية الـ parameters
)

ContestInfoSectionImproved(
  titleCtrl: _titleCtrl,
  descriptionCtrl: _descriptionCtrl,
  totalPrizesCtrl: _totalPrizesCtrl,
  committeesCount: _committeesCount,
  onCommitteesChanged: (v) { if (v != null) setState(() => _committeesCount = v); },
  isMobile: isMobile,
  primaryColor: widget.primaryColor,
)

ScheduleSectionImproved(
  days: _days,
  onAddDay: () => setState(() => _days.add(DayBlock())),
  onRemoveDay: (i) => setState(() => _days.removeAt(i)),
  pickDate: _pickDate,
  primaryColor: widget.primaryColor,
  hasIncompleteDays: _hasIncompleteDays,
  hasOverlaps: _hasOverlaps,
  isMobile: isMobile,
)
```
